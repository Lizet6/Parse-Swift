Subscription

open class Subscription<T> : QueryViewModel<T>, QuerySubscribable where T : ParseObject

A default implementation of the QuerySubscribable protocol. Suitable for ObjectObserved as the subscription can be used as a SwiftUI publisher. Meaning it can serve indepedently as a ViewModel in MVVM. Also can be used as a Combine publisher. See Apple’s documentation for more details.
