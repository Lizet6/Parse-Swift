ParseGitHub

public struct ParseGitHub<AuthenticatedUser> : ParseAuthentication where AuthenticatedUser : ParseUser

Provides utility functions for working with GitHub User Authentication and ParseUser‘s. Be sure your Parse Server is configured for sign in with GitHub. For information on acquiring GitHub sign-in credentials to use with ParseGitHub, refer to GitHub’s Documentation.
